I've designed everything from a 50×70cm poster to a 12-meter building wrap. The design principles are consistent across those scales, but the practical requirements are not. Large format print has its own set of constraints that need to be understood before you design, not discovered during production.

## Viewing Distance Changes Everything

The fundamental difference in large format design is viewing distance. A standard A2 poster is read at arm's length or slightly further. A billboard is read from 50 to 200 meters away. The design decisions that serve close viewing fail at distance, and vice versa.

At distance, fine detail disappears. Thin type strokes merge with the background. Subtle color variations become invisible. Complex imagery becomes unreadable texture. Everything simplifies into mass, shape, and value — the same things that matter in thumbnail tests for smaller formats, but in a much more extreme way.

For large format work, I design with maximum simplicity. One or two messages. Bold, heavy type. High contrast. Simple, reduced imagery. Elements that read clearly as silhouettes. If it doesn't work as a simple, bold composition, it won't work at billboard scale.

## Image Resolution for Large Format

The resolution requirements for large format print are counterintuitively low — and this trips up designers moving to large format from standard print work.

For a poster printed at 300 DPI, a standard resolution, an A2 print requires roughly 2500×3500 pixels. For a 6-meter wide billboard printed at 72 DPI (which is typical for large format digital print because viewing distance is so great), the file needs to be around 17,000 pixels wide. These are very different files with very different preparation requirements.

The DPI standard for large format varies by application:
- Vehicle graphics: 100-150 DPI at print size
- Exhibition displays viewed at 2-3m: 150 DPI
- Outdoor billboards: 15-72 DPI depending on size and minimum viewing distance

I always confirm the printer's specification before building the file. Providing a file that's the wrong resolution — usually too low — is the most common large format production problem.

## Material and Finish Considerations

Large format print goes on surfaces that standard print doesn't: vinyl banners, window film, rigid foam board, fabric, building wraps, floor graphics. Each material has different color behavior, different surface texture, and different lamination or finishing requirements.

Colors look different on vinyl than on paper. Adhesive films have minimum radius curves for outdoor applications. Some inks aren't UV-stable and fade outdoors. Fabric banner materials have a woven texture that's visible at close range and affects fine detail.

I specify materials with the same care I bring to paper stock for standard print work. The material is part of the design.

## Type at Large Format Scale

The rules for type that apply in poster work apply even more strictly in large format. No thin weights — they disappear at distance. No complex typefaces with fine detail — the detail is lost. No dense type that requires close reading — viewers are in motion.

Large format type should be heavy, simple, and large enough to read from the maximum intended viewing distance. I test this with a calculation: at 100 meters, a letter needs to be approximately 25cm tall to be legible. At 50 meters, 12-15cm is typically sufficient.

These numbers feel enormous in the design file. They look correct at actual scale.
