Every design program teaches Gestalt principles. Proximity, similarity, continuation, closure, figure-ground. Students learn the names, pass the exam, and then largely forget about them in practice. Which is a shame, because these principles describe real perceptual phenomena that directly govern how viewers experience design.

## Proximity: The Grouping Principle

Elements that are close together are perceived as related. This is the most immediately actionable Gestalt principle for layout work. When I want two pieces of information to be understood as connected — a caption and its image, a headline and its subhead — I put them close together with a gap between that group and the next group.

When elements are evenly spaced, they don't group — they read as a list. Intentional spacing differentials are how you create the groupings that communicate relationships.

## Similarity: Building Visual Systems

Elements that look similar are perceived as belonging together. This is the perceptual basis for visual systems: if all the body text in a layout is the same typeface, size, and weight, the viewer understands it as a category. If section headers all share a treatment, they read as a navigational system.

Similarity failures produce confusion. A composition where three elements share a treatment but one is slightly different signals unintentional inconsistency rather than hierarchy. If you want an element to stand out, make it comprehensively different — not subtly different.

## Closure: The Mind Completing the Form

The viewer's mind completes incomplete shapes. A circle with a small gap reads as a circle. A square made of four dots reads as a square. Designers use closure to create marks that are implied rather than explicit — which can create visual interest and cognitive engagement.

Closure works in logo design when the incomplete shape is clear enough that the completion is immediate and unambiguous. If the viewer has to work to complete the form, the mark reads as broken rather than clever.

## Figure-Ground: The Fundamental Tension

Every composition creates a relationship between figure (what we perceive as the object) and ground (what we perceive as background). This relationship is never neutral — the design is always pushing the viewer to read certain elements as figure and others as ground.

When figure-ground relationships are ambiguous, the viewer's eye flickers between readings — which can be used as a deliberate effect or can result in an unstable, confusing composition. I'm deliberate about which elements I want read as figure and I use contrast, scale, and color to enforce that reading.

## Using These Tools

I don't think through Gestalt principles consciously for every decision. But when a composition isn't working and I can't immediately identify why, Gestalt gives me a diagnostic vocabulary. Is a grouping unclear because proximity is ambiguous? Is hierarchy broken because similarity is inconsistent? Is a mark unstable because the figure-ground relationship is fighting itself? The principles are a troubleshooting framework as much as a design guide.
