## Icons Are Part of the Brand Too

Most brand identity projects include typography, color, and a logo. The icon set is often an afterthought — grabbed from a stock library, downloaded from an icon pack, or produced by a different person using different rules.

The result is a brand system where the logo feels considered and the icons feel generic. The icons are used constantly — in navigation, in communications, in presentations — and their generic quality bleeds into the overall brand impression.

A custom or customized icon set, designed to match the brand's visual character, is one of the most high-leverage investments a brand system can make.

## The Visual DNA of an Icon Set

Every well-designed icon set has a consistent visual DNA — a set of design decisions that apply to every icon in the set:

**Stroke weight** — how heavy are the lines? The stroke weight should relate to the weight of the brand's typefaces. A heavy-weight brand should have heavier icons.

**Corner radius** — are corners sharp or rounded? This is a strong personality signal. Sharp corners feel technical and precise. Rounded corners feel approachable and friendly. The corner radius should match the brand's personality and any corner treatment in the logo or type.

**Line vs. fill** — are icons outline-only (stroked) or solid (filled)? Outlined icons feel lighter and more refined. Filled icons feel more solid and immediate. Many systems use both at different size tiers.

**Optical sizing** — icons used at 16px need different levels of detail than icons used at 48px. The same file won't work at both sizes. I design icons at multiple key sizes with appropriate detail for each.

**Grid and proportion** — all icons in a set should be drawn on the same grid, with the same optical size within the containing bounding box. Icons that share a grid look like a set. Icons drawn ad hoc at different scales look like a collection.

## Matching the Identity

I pull the brand's design DNA into the icon set explicitly. If the logo uses a specific corner radius, I use that radius. If the brand typeface has distinctive stroke terminals, I create icon terminals that echo them.

These connections don't need to be obvious. They create subliminal cohesion — the icons feel like they belong to the same world as the logo and type even if the viewer can't articulate why.

## How Many Icons Is the Right Number

One of the most common mistakes in icon set design is over-producing. A set of 300 icons, most of which will never be used, is more work for everyone involved in maintaining the system.

I start with a core set: the 20–30 icons that the brand will use in its primary contexts. Navigation icons, common action icons, category icons for the brand's content. That's the first milestone.

Additional icons are added as specific needs arise, designed to the same DNA as the core set.

## Documentation for Icon Systems

Icons need to be documented with the same care as other brand elements: what the icon means (not just what it looks like), at what sizes it should be used, what variations exist, what it should not be used for.

Without this documentation, icons get used in ways that corrupt their meaning. An icon designed to indicate "settings" gets repurposed to mean "tools" or "preferences" by different teams, creating inconsistency that erodes the system.

Name the icons. Document their uses. Protect their meaning.
