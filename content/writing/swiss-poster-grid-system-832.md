## Why Swiss Posters Still Hold Up

I keep a folder of Swiss International Style poster references from the 1950s and 60s. Not as historical curiosities — as working reference. Armin Hofmann, Josef Muller-Brockmann, Emil Ruder. The work is sixty years old and it communicates more clearly than most contemporary posters I see.

The reason isn't nostalgia. It's structure. These designers built a systematic approach to organizing information on a page that is essentially correct. The specific aesthetic has been evolved and subverted many times over, but the underlying structural logic is still sound.

## The Modular Grid

The Swiss approach to grid design is modular: the page is divided into equal units both horizontally and vertically, creating a grid of cells rather than just columns. Content is positioned within and across these cells, creating a consistent spatial logic that unifies the page.

The modular grid is more restrictive than a column grid but produces more spatial coherence. When elements span multiple modules, the size relationships between them are predetermined by the grid — not arbitrary. Everything on the page is related to everything else through the common module.

## White Space as Structural Element

The Swiss designers treated white space not as empty area but as a structural component of the grid. Space between elements follows the same modular logic as the elements themselves. A gap of one module between elements creates a different relationship than a gap of two modules or half a module.

This systematic approach to spacing is what gives Swiss-influenced work its characteristic clarity. The reader's eye doesn't have to work to find order — the order is built into the structure.

## Type Within the Grid

In Swiss poster work, type is positioned within the grid with the same discipline as any other element. Type blocks align to the module, baseline grids align to the horizontal unit. Type doesn't float freely — it occupies defined positions within the spatial logic.

This constraint seems limiting but produces enormous compositional confidence. When every element is in a grid-logical position, the composition reads as considered rather than assembled.

## Applying This Today

I don't apply Swiss grid logic slavishly. I apply the underlying principle: establish a spatial system before placing content, and let the system guide every placement decision.

The specific module size, the number of columns, the margin proportions — these vary by project and output format. But the discipline of committing to a spatial system and then executing within it, with intentional breaks rather than random ones, produces the same quality of result that made those 1960s posters timeless.

The grid isn't a historical artifact. It's a thinking tool.
