Animating a logo is not the same as making a logo move. That distinction sounds pedantic until you've watched a beautifully crafted static mark get ruined by an animation that fights the design rather than extending it.

Logo animation has one job: reveal or reinforce the character of the mark. Every motion choice should serve that goal.

## Starting With the Static Logic

Before I touch After Effects, I spend time re-analyzing the static mark. What is the mark doing? What makes it distinctive? What story does the geometry tell?

A mark built on circular geometry has inherent rotational or radial logic. An animated reveal that draws it as a rotation acknowledges that logic. A mark built on directional tension — forms that suggest movement in a specific direction — has an animation direction built into it. Following that logic makes the animation feel inevitable.

Ignoring the static mark's internal logic and applying a generic "fly in and fade up" animation produces work that could be applied to any logo, which means it belongs to none.

## The Reveal vs. the Loop

Logo animations typically serve two purposes: a reveal (played once when the brand appears) and a loop (played continuously in a branded context, like a loading screen or broadcast bug).

These require different approaches. A reveal can be longer and more elaborate — it's introducing the brand, so time spent building the mark has narrative purpose. A loop needs to be seamless, subtle, and non-distracting — it's a presence, not a performance.

I design reveal and loop variants separately, even for the same mark. They're different problems.

## Duration and Restraint

Logo animation should be brief. A reveal that takes longer than 3 seconds is usually too long. The mark should arrive with enough build time to feel deliberate but not so much that it feels slow or self-indulgent.

I default to 1–2 seconds for most logo reveals and test against the contexts where they'll live. A broadcast intro can justify 3 seconds. A website loading animation should be well under 2.

## Sound Design

Motion without sound is incomplete in contexts where audio is expected. A logo reveal with appropriate sound design has a different quality than the same animation without sound.

I don't do the sound design myself for most projects — I collaborate with audio designers or provide direction. But I think about the sonic character of the mark when designing the motion: what does this mark sound like? Is it percussive, tonal, noisy, or silent? The motion and sound should arrive from the same brand character.

## Technical Delivery

Logo animations need to be delivered in formats appropriate to their contexts: After Effects source files for broadcast, transparent video (ProRes with alpha) for video applications, Lottie JSON for digital and web contexts, GIF for fallback where nothing else works.

Each format has different quality characteristics, file size constraints, and transparency support. I specify the delivery format based on the intended use before I start animating, because format constraints can affect the animation design itself.
