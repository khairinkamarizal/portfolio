The most common failure mode when combining photography and typography in poster design is that the two elements look like they were designed separately and then stacked. The photo is the photo. The type is the type. They coexist but don't integrate. The result feels like a compromise rather than a design.

Real integration — where type and image become a single visual system — requires decisions about both elements simultaneously, not sequentially.

## The Zones Problem

Most designers start with an image and then find space for text. The image occupies a zone; the text occupies a different zone. This produces the layered, disconnected look I'm describing. The image and type share a rectangle but nothing else.

Integration requires that the text respond to the image. Where the image has tonal weight, the type can use that weight. Where the image has empty sky or open background, the type can inhabit that space. The type finds its position inside the image logic rather than outside it.

## Reading the Image Before Placing Type

My process: before placing a single character, I study the photograph for compositional opportunities. Where are the natural resting zones — areas of relative tonal simplicity where type can sit without being destroyed by competing texture? Where are the strong lines — architectural edges, horizon lines, strong shadows — that type can align with? Where does the image's focal point sit, and does the type placement reinforce or counterpoint that?

I also look at the tonal values of the image and think about type color. Light type on dark areas of the image. Dark type on light areas. Or, more interestingly, color type pulled directly from a hue within the image itself, which creates a visual echo that unifies the two elements.

## Techniques for Integration

Clipping type to image elements — having a photograph visible through letterforms — can create strong integration, though it's a technique I use selectively because it can feel like a visual trend rather than a conceptual choice.

Running type that responds to the contours of the image — following the line of a skyline, responding to the edge of a figure — is subtler and often more elegant. The type is shaped by the image rather than imposed on it.

Type that breaks the image plane — a headline that starts behind a foreground element and continues in front of it — creates depth and establishes that text and image occupy the same three-dimensional space.

## Color Extraction vs. Contrast

I use two different color strategies depending on the goal. If I want text and image to feel unified, I extract a color from within the image for my type. The eye perceives the type as growing from the image rather than sitting on top of it.

If I want the type to command authority and stand independent of the image, I use maximum contrast — white on dark, black on light — which announces the text as a distinct layer. This is the right choice when the text needs to win the hierarchy battle.

## Testing the Integration

The integration test I use: if I cover the text, does the image composition still feel complete? If I cover the image, does the text arrangement still feel considered? The ideal is that both hold up independently, but together they create something neither could achieve alone.

If either half collapses without the other, the integration is incomplete. They should be mutual, not dependent.

Photo and type speak different visual languages. Integration is translation work — making them say the same thing together.
