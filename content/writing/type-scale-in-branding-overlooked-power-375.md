The type scale — the specific sizes at which type is set across a brand system — is one of the most consequential typographic decisions you can make, and one of the least discussed. Most designers choose sizes by feel, accumulating arbitrary values that work individually but create no coherent system.

## What a Type Scale Does

A type scale is a set of proportionally related sizes derived from a base value and a ratio. All sizes within the scale have mathematical relationships to each other. When applied consistently, this creates a rhythmic quality to text — elements at different levels of hierarchy feel like they belong to the same visual language rather than being independently sized.

Beyond aesthetics, a type scale enforces discipline. When every size decision is constrained to the scale, it's impossible to accidentally create two heading levels at nearly identical sizes. The scale provides built-in visual differentiation.

## Choosing the Ratio

The ratio determines how dramatic the size steps are. Small ratios (1.125 to 1.25) produce subtle distinctions — appropriate for editorial text design where hierarchy is nuanced. Large ratios (1.5 to 1.618) produce dramatic distinctions — appropriate for brands that need strong, assertive hierarchy.

I choose the ratio based on the brand's personality and content requirements. A brand that communicates with authority and clarity needs strong scale separation. A brand built around detail and nuance might use a tighter scale with more intermediate values.

## The Base Size

The base size is the body copy size — the size at which the primary text in the system is set. Everything above it (headings, display) scales up from this base. Everything below it (captions, footnotes) scales down.

For digital brand applications, I typically use 16px as the base. For print brand applications, the base depends on the primary document type — a brochure might use 9pt, a report might use 10pt.

## Scale as Brand Voice

Here's what's underappreciated: the specific scale values a brand uses become part of its visual voice. A brand that uses a generous scale with large display type and relatively small body type communicates assertiveness and visual hierarchy. A brand that uses a compressed scale with moderate display and comfortable body type communicates accessibility and approachability.

These are real differences in how the brand feels. The type scale is a personality-setting decision as much as a functional one.

## Documenting the Scale

Brand guidelines should include the complete type scale — every value, what it's used for, and the usage context. Not just "use 24pt for subheadings" but the full scale table with all values and their relationships.

This documentation allows anyone working in the brand system to extend it consistently. A new content type that needs a size not in the guidelines can be derived from the scale's ratio rather than chosen arbitrarily.
