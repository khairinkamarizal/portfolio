## The photograph can do most of the work

When a strong photograph is available — one with genuine visual power, clear subject, and emotional resonance — the designer's primary job is to not ruin it. Get out of the way. Let the image work. Add only what the image can't do alone.

This sounds simple and is actually difficult. The instinct when designing with a strong image is to add — more type, more graphic elements, more complexity. Resisting that instinct requires confidence in the image's ability to carry the communication.

## What photography does that type can't

A great photograph does several things that typography alone cannot: it creates immediate emotional connection through recognition of human expression and situation. It provides specific, credible visual evidence — this event happened, this person exists, this place is real. It communicates sensory experience — texture, light, temperature — that abstract design elements only approximate.

Understanding what the photograph is doing in the poster tells me what the type needs to do in addition. If the image communicates the emotional register fully, the type needs only to provide information (event name, date, location). If the image provides context but not emotional direction, the type carries more weight.

## Placement relative to the image's visual structure

Strong photographs have internal visual structure — lines of force, areas of tension and release, dominant focal points, secondary elements. Type placement that works with this structure feels natural and integrated. Type placement that ignores it feels arbitrary.

I study the photograph's composition before placing any type. Where does the eye enter? Where does it travel? Where does it rest? The type should participate in this movement, not interrupt it.

## The cropping decision

The same photograph, cropped differently, is a different image with different compositional properties. Cropping is a significant design decision — one that changes the image's relationship to the type and the poster's overall composition.

I explore multiple crop options before settling on one. Tight crops that emphasize the subject. Wider crops that give the subject environmental context. Asymmetric crops that create more interesting negative space for type. The crop that came from the photographer is a starting point, not a final answer.

## Full-bleed vs. contained photography

A full-bleed photograph — one that extends to every edge of the poster — creates a specific visual experience: the viewer is immersed in the image space. Type floats in that space rather than sitting beside the image.

A contained photograph — one that exists within the poster's compositional structure with clear type zones — creates a different experience: the image and the type are distinct elements in dialogue. Both approaches work; the choice should be made based on what the specific image and the specific communication goal require.

## Color continuity between image and type

When type floats over or adjacent to a photograph, the color relationship between the type and the image's color palette affects how well they integrate. Type that picks up a color from the image's own palette creates continuity. Type in a completely unrelated color creates contrast.

I usually sample from the image's palette for headline color, then test whether the sampled color provides sufficient contrast for legibility. If the best-integrating color doesn't provide adequate contrast, I modify its value (lightness/darkness) until it does both.
