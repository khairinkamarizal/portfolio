Every poster design I produce gets a production check before it goes to print — not just a preflight for file correctness, but a deliberate review of whether the typographic choices will survive the specific print process and stock specified. Typography that looks refined in Illustrator at 100% on a calibrated screen is not the same as typography that reads correctly on an A1 poster printed offset on uncoated 170gsm.

The variables are real, and ignoring them produces disappointing results.

## Weight and Stroke at Scale

Thin type strokes are the first casualty in production. On uncoated stock, ink absorbs into the paper fibers and spreads slightly — the dot gain effect — which means fine strokes get heavier and can fill in. At 100pt, a hairline stroke in a high-contrast typeface is fragile. At 12pt, it may become illegible.

My rule: nothing lighter than a regular weight for type that needs to be legible at any scale. For body and supporting copy on posters — usually set between 8pt and 14pt — I go to medium or semi-bold if the stock is uncoated.

For display and headline type at large sizes (48pt and above), thin weights can work on coated stock but need to be tested on uncoated. I output a printed proof at the intended size before committing.

## Reversed Type and Fill-In

Reversed type — white or light type on a dark background — has the opposite problem from thin type on light backgrounds. The dark ink spreads slightly toward the lighter type, narrowing the counter spaces and thin strokes.

The effect is most visible in small reversed type: the apertures of letters like 'e', 'a', and 's' can start to close up. The counters of 'o', 'p', 'd', 'q', 'b' narrow.

Compensation: use a slightly heavier weight for reversed type than I'd use for the equivalent positive setting. Increase the type size if possible. Choose typefaces with generous apertures and open counters for any reversed text that needs to be legible.

## Color Type Over Dark Backgrounds

Colored type on dark backgrounds is another context where weight matters. A brand blue or deep red type on a black background is technically reversed type, and the same fill-in effects apply. Additionally, if the specified ink color is close in value to the background, contrast may be insufficient for legibility.

I always check value contrast in grayscale before finalizing any colored type over dark backgrounds.

## Script and Decorative Typefaces at Small Sizes

Script typefaces and highly decorative display faces often have strokes that approach hairline at their thinnest. These faces are designed for large display use. At small sizes — say, below 24pt — the hairline strokes can print incorrectly or disappear entirely.

My practice: never use a high-contrast script or decorative face below 36pt without a production proof. And never use them at all for any text that needs to be legible rather than merely ornamental.

## The Proof Is the Answer

No amount of on-screen evaluation fully substitutes for a proof at actual print size on the actual stock. Before finalizing the typographic choices in any poster that's going to print, I pull a proof. If the type looks correct, the choices are confirmed. If anything looks wrong — too thin, too cramped, filling in — I adjust before the job goes to press.

The proof is not an optional step. It's the last line of defense between design intent and production reality.
