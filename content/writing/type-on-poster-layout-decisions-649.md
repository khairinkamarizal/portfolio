Typeface selection gets a lot of attention in poster design discussions, and it deserves it. But the layout decisions — how type is sized, positioned, spaced, and related to other elements — matter at least as much as what typeface you choose. Excellent type handled poorly produces a poor poster.

## The Hierarchy Question

Before I place any type, I answer: what is the most important thing this poster communicates, and what's the order of everything else? This hierarchy question has to be answered editorially — not by the designer's aesthetic preference, but by what the viewer needs to know and in what order.

For most event posters: the event or performer name is primary. The date and venue are secondary. Everything else is tertiary.

Once the hierarchy is established, every layout decision reinforces it or undermines it. I review layouts constantly against the hierarchy question: does the visual weight of each element correspond to its informational importance?

## Line Length and Legibility

The line length of display type — the number of characters per line in a headline or primary typographic element — affects both legibility and compositional energy.

Long lines create horizontal reading rhythm. They read calmly, comprehensively. Short lines create vertical stacking — a different rhythm, more staccato, more emphatic per line.

For event posters, I often break headlines into short line fragments — sometimes a single word per line — to create a stacked typographic form with strong vertical presence. Each line reads as a beat.

## Leading at Display Scale

Leading (line spacing) at display sizes requires different logic than at text sizes. The rule of thumb for body text — leading at 120-145% of type size — produces too much space between display-size lines. The lines feel disconnected rather than related.

At large display sizes, I reduce leading significantly — sometimes to 90% of type size or less — to pull lines together into a tight block. The block reads as a unified typographic mass, not as individual lines.

This tight leading only works if the descenders of one line don't crash into the ascenders of the line below. I always check for this collision at final sizing.

## Alignment: A Decision, Not a Default

Left-aligned type, centered type, right-aligned type — each has a different feeling and implies a different compositional logic.

Centered type is formal and symmetrical. It suggests balance, occasion, tradition. It's appropriate for formal announcements, classical music events, institutional communications.

Left-aligned type is direct and editorial. It's the alignment of reading — the eye knows where each line starts. It works for most contemporary event work.

Right-aligned type is less common and more distinctive. It creates a strong flush-right edge and a ragged left margin, which can be used compositionally. I use it when I want the type to anchor to the right side of the frame.

The mistake is defaulting to centered because it seems "safe." Centered alignment is a specific choice with specific implications. Make it deliberately.

## Breaking the Alignment Rules

I break alignment rules deliberately. A single word that steps out of an otherwise flush-left alignment reads as intentional emphasis. A word set at a different angle than the rest of the text — particularly in display work — creates graphic energy. Rules are tools; knowing when to break them is part of the skill.
