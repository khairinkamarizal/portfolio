Most color palettes I see in brand guidelines are lists. Five colors with their HEX codes. No hierarchy, no usage rules, no understanding of what role each color plays. The list is necessary but insufficient — what makes a color palette a system is understanding how the colors relate to each other and how they're intended to be used.

## The Palette Structure

A well-designed brand color palette has a clear internal structure:

**Primary brand color(s):** One or two colors that are definitionally the brand. These are the colors that, when someone sees them, they think of this brand. Primary colors should be used consistently, prominently, and protectively — not diluted by being used interchangeably with secondary colors.

**Neutral(s):** A neutral tone — almost always needed — that provides the background and supporting structural color in most brand communications. The neutral interacts with the primary color more than any other color in the palette. Getting the neutral right is as important as getting the primary right.

**Secondary/accent colors:** A small set of supporting colors used for specific purposes — to create contrast in data visualization, to differentiate product lines, to add visual range in communications. Secondary colors should be clearly subordinate to the primary palette in usage frequency.

**Functional colors:** In some brand systems, specific colors are reserved for functional roles: success states, warning states, error states in digital contexts, or specific product category indicators. These have defined jobs and shouldn't be repurposed.

## The Relationships Within the Palette

A good palette works as a system because the colors are in designed relationship with each other. This means:

The primary color and the neutral should create a coherent atmosphere when used together — which is most of the time. A warm primary color paired with a cool neutral creates productive tension; a warm primary with a warm neutral creates warmth but can feel monotonous; a cool primary with a cool neutral feels clean and minimal.

The accent colors should complement the primary without competing with it. An accent color that's as visually strong as the primary creates two equally dominant colors — which is one too many.

## Usage Proportions

The colors in a palette aren't meant to be used in equal measure. A helpful way to think about it: the primary and neutral make up the majority of any communication's visual surface (roughly 70-80%). The secondary/accent colors appear in smaller doses for emphasis and distinction (roughly 20-30%). Using all colors in roughly equal amounts creates visual noise where nothing is primary.

I document usage proportions in brand guidelines alongside color specifications. Not precise percentages — that would be over-prescriptive — but clear language about which colors are dominant and which are accents.

## Testing the Palette as a System

Before finalizing a palette, I test it as a system:

- Set a paragraph of text in the primary color on the neutral. Does it read clearly?
- Create a simple composition using only the primary and neutral. Does it feel like the brand?
- Add an accent color element. Does it feel subordinate and appropriately emphatic, or does it compete?
- Combine all palette colors in a single composition. Does the combination feel coherent?

A palette that passes these tests is a system. One that fails is still just a list.
