If I had to identify the single typographic variable that separates competent print design from genuinely good print design, it would be leading. Not typeface selection. Not size. Leading — the space between lines.

It's the variable most designers set once at the start of a project and then never revisit. It gets treated as a background decision. That's a mistake, because leading determines whether a page breathes or suffocates, whether a long block of text feels readable or exhausting, whether the typographic texture of a layout is even or ragged.

## The Mechanical Problem with Default Leading

Most design software sets default leading at 120% of the type size — 12pt type gets 14.4pt leading. This is a reasonable baseline for body text at standard sizes in standard conditions, but it's a baseline, not an answer.

Default leading is too tight for longer-measure text (where the eye needs more separation to find the next line), too loose for narrow columns, and often wrong for display type, where optical adjustments are always necessary.

I reset leading manually on every project based on the actual conditions: typeface x-height, column measure, text length, and reading context.

## X-Height Changes Everything

Two typefaces set at 14pt with identical leading will look completely different if their x-heights differ. A tall x-height (like many geometric sans-serifs) produces text that looks tight because the uppercase and lowercase letters fill more of the em. A low x-height (like many old-style serifs) leaves more apparent space between lines.

I calibrate leading to the visual density of the typeface, not just to the point size. If a typeface feels tight, I add leading even if the numbers suggest it should be adequate.

## The Baseline Grid Connection

In multi-column print layouts, leading and the baseline grid need to be in sync. My baseline grid is always set to the same value as my body text leading. This ensures that text set in different columns shares a baseline, which creates visual alignment that readers register as calm and resolved even if they can't explain why.

When columns don't share a baseline — when text is misaligned across the gutter — it creates a subtle restlessness on the page. It's not legibility-breaking, but it undermines the quality impression of the work.

## Display Type Leading Rules

For headlines and display type, the default leading is almost always too generous. A headline set at 72pt with 86pt leading will have awkward, vacancy-signaling gaps between lines. I usually set display leading at about 90–95% of the point size, sometimes tighter for condensed or uppercase-only settings.

I also apply manual tracking adjustments per pair of lines when a headline breaks across three or more lines, because automatic leading doesn't account for the visual weight of specific line breaks.

## Testing the Rhythm

The test I use: set a long paragraph and read it aloud at the pace you'd normally read. If you have to consciously hunt for the next line, leading is too tight. If the text feels spread out and disconnected, it's too loose. The right leading is the one where the reading is unconscious and continuous.

This is a physical, embodied test, and it works. Trust your reading experience over the numbers.

Leading sets the tempo of the text. Get it right and the reader doesn't notice it. Get it wrong and they can't finish the page.
