## The Sub-Brand Challenge

A sub-brand occupies a defined space within a larger brand architecture. It needs to be related to the parent brand — sharing recognizable DNA — while also being distinct enough to serve a specific audience or product category that the parent brand can't fully address.

This is a different design problem from building a brand from scratch. The constraints are tighter and the relational logic is more complex.

## Reading the Parent Brand

Before designing a sub-brand, I study the parent brand's identity system deeply. What are its defining visual characteristics? What's the geometric logic of the mark? What does the color system communicate? What typefaces and their proportions define the brand's typographic voice?

The sub-brand has to make design decisions that are responsive to all of this. It's designing in relationship, not in isolation.

## Shared and Differentiated Elements

The sub-brand system needs clarity about what's shared with the parent brand and what's differentiated. Typically, the typographic system is shared — using the same typeface family creates family resemblance across the brand architecture. Color is often the primary differentiator — the sub-brand uses a distinct color that's clearly within the same color sensibility as the parent but different enough to be distinguishable.

The mark relationship depends on the architecture model: an endorsed brand uses a version of the parent mark with modification; an independent sub-brand may have its own separate mark.

## Maintaining the Design Logic

The parent brand's design logic — the geometric principles, the proportional relationships, the formal character of its mark — should be legible in the sub-brand's design decisions. If the parent brand is built on strict geometric construction, the sub-brand should be too. If the parent has organic, humanist qualities, the sub-brand should reflect that.

This is not about making the brands look the same. It's about making them look related — like they come from the same family, even if they're visually distinct.

## The Endorsement Relationship

Some sub-brands carry explicit parent brand endorsement: "by [Parent Brand]" or "a [Parent Brand] company" in a lockup with the sub-brand name. This endorsement relationship needs to be designed carefully — the relative size and placement of endorsement versus primary mark, the typographic treatment of the endorsement text.

Endorsement layouts that look like they're fighting each other signal an unresolved brand architecture. They should look designed.

## Preventing Architectural Drift

Sub-brands, once released into the world, generate more sub-brands. Product lines within the sub-brand. Campaign extensions. Related initiatives. Without a clear architectural framework from the beginning, each of these generates ad-hoc decisions that accumulate into visual chaos.

I build sub-brand design work with an eye toward the next generation of decisions it will generate, documenting the rules for how the architecture can be extended.
