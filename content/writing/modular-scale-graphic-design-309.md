Random size choices accumulate into visual incoherence. When a layout has type at 9, 12, 14, 18, 23, and 36 points — chosen because each one looked about right — the result is a collection of decisions rather than a system. Modular scale eliminates this by deriving all size decisions from a single ratio applied repeatedly to a base value.

## What a Modular Scale Is

A modular scale is a sequence of numbers with a consistent ratio between each step. The most commonly used ratios in design: 1.25 (major third), 1.333 (perfect fourth), 1.414 (augmented fourth), 1.5 (perfect fifth), 1.618 (golden ratio).

Starting with a base size of 16pt and a ratio of 1.333: 16, 21.3, 28.4, 37.9, 50.6. Every size in the scale has a harmonic relationship to every other size. Elements at different scales feel proportionally correct in relation to each other.

## Applying the Scale

I pick a base size from my content requirements — usually the body text size — and then select a ratio that gives me the range I need. If I'm designing a poster with one large headline, a subtitle, and body copy, I need about three scale steps. The golden ratio (1.618) gives me dramatic size contrast. The major third (1.25) gives me more gradual steps.

Once the scale is set, every size decision in the project maps to a scale value. I don't choose sizes that fall between scale values unless there's a specific reason. The constraint is generative — it forces size choices to be systematic rather than arbitrary.

## Combining Two Scales

For complex multi-level hierarchies, I sometimes use two overlapping scales — one for the primary content hierarchy and one for supporting elements. Both derive from the same base, with different ratios. The resulting system has more values to work with while still maintaining harmonic relationships.

This approach works well for editorial print design where you're managing multiple content types: primary text, pull quotes, captions, footnotes, headers, subheaders, page numbers.

## Beyond Typography

Modular scale applies to spatial decisions as well as type sizes. Margin widths, column gutters, image dimensions — all can be derived from the same scale that governs the type. When the spatial system and the typographic system share a common ratio, the layout has a coherence that's felt even when it can't be articulated.

I don't use modular scale as a rigid system that overrides visual judgment. I use it as a discipline that requires me to have a reason for any size that falls outside the scale. That discipline alone eliminates most arbitrary decisions.
