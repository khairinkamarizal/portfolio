Print errors are permanent. Unlike digital work where a mistake can be updated after publication, a printed poster with an error exists as a physical object that costs money to reprint and causes pain for everyone involved.

A thorough proofing process is one of the most valuable habits a print designer can develop.

## The Proofing Mindset

Proofing is not design. When I'm in proofing mode, I'm not evaluating whether the design is good — I've already made those decisions. I'm systematically checking whether the file is correct and complete, whether the design is correctly executed, and whether there are any errors that need to be caught before the file goes to print.

This requires a different mental mode than designing. Design requires creative judgment. Proofing requires systematic verification.

## The Technical Checklist

Before any file goes to print, I check:

**Bleed:** Does the design extend to the bleed edge on all sides? No white edges from inadequate bleed.

**Safe zone:** Is all critical content (especially type) inside the safe zone? Nothing important near the trim edge.

**Resolution:** All raster images at 300 DPI or higher at final print size. No upsized low-resolution images.

**Color mode:** CMYK, not RGB. No RGB images or objects embedded in the file.

**Black:** Is large black text or areas specified as rich black or 100K? Rich black (CMYK composite) for large elements, 100K for small text to avoid registration issues.

**Overprint settings:** Are any objects incorrectly set to overprint? Is black text set to overprint (correct) and colored text not set to overprint (usually correct)?

**Fonts:** Are all fonts outlined or embedded? No missing fonts.

**Links:** All linked images present and up to date. No modified or missing links.

**Spelling:** Every word proofread. Every number verified. Every URL checked.

## The Human Eye Check

After the technical checklist, I print a copy at full size (or as close to full size as possible) and check it by hand:

- Read every line of type aloud (catches transpositions, missing words, repeated words)
- Check every number specifically (dates, times, prices)
- Verify that names are spelled correctly
- Check contact information is accurate

Reading aloud is more effective than reading silently because it forces engagement with every word rather than allowing the eye to fill in expected content.

## The Client Proof

For any file with significant text content — especially names, dates, prices, or contact information — I send a proof to the client for sign-off before going to print. Client approval puts the responsibility for factual accuracy appropriately on the party who owns the information.

The client proof should be clearly marked as "proof for approval — please check all text content" and should require a specific written sign-off, not just silence.

## The Hard Stop

If I'm uncertain about anything in a file — a measurement that doesn't look right, a color that seems off from what was specified, a font that looks different from the approved version — I stop and verify before sending to print. The cost of a short delay to verify is always lower than the cost of reprinting.

The hard stop is a professional habit that has saved me from expensive mistakes more times than I can count.
