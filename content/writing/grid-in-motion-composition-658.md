When I moved from static design to motion, one of the biggest adjustments was understanding how grid logic works in a moving composition. In static design, the grid is a structural framework you design within. In motion, the grid is still there — but elements move across it, through it, between positions on it, and the grid structure has to remain legible through the animation.

## The Static Grid as Motion Foundation

I build every motion composition on the same grid I'd use for a static layout. The grid establishes the safe zones, the structural positions, the spatial relationships. Elements start and end on grid positions — their in-motion path is between grid-anchored states.

This matters because motion that ends at random positions reads as unsettled. When an element arrives at a position and that position has clear relationship to the grid, the arrival feels intentional. The element is where it belongs.

## Compositional Zones in Motion

For motion work that contains multiple elements appearing and disappearing, I define compositional zones — areas of the frame that are "owned" by specific types of content. The headline zone. The supporting information zone. The brand identity zone.

Elements within each zone move and animate within that zone. They don't cross into other zones unless there's a specific compositional reason. This creates predictability in the composition — the viewer learns where to look for different types of information as the piece develops.

## Entry and Exit Points

Where elements enter and exit the frame is a grid decision. I define a limited set of entry and exit positions — usually the edges of the safe zone, aligned to grid columns — and route all element movements through these points.

This limitation sounds like it would make all animations look the same. In practice, it creates a coherent spatial vocabulary that makes the piece feel architectural rather than arbitrary.

## The Stable Frame

Not every element in a motion piece should be moving at all times. In fact, some of the most effective motion compositions have a stable, fixed background structure that remains constant throughout, with content entering and exiting against that stable background.

The stable frame gives the viewer spatial orientation throughout the piece. They know where they are because the structural elements haven't moved. The entering and exiting content is the news; the stable frame is the newspaper's layout.

## Multi-Screen and Format Adaptations

Brand motion work often needs to work across multiple formats: landscape for YouTube, square for Instagram, portrait for Stories. The grid for each format is different, but the spatial logic should be consistent.

I build multi-format motion systems by defining the compositional logic first — the relationships between elements, the zone structure, the entry and exit vocabulary — and then adapting that logic to each format rather than redesigning from scratch for each one. The work is in the system, not in the per-format execution.

## When to Break the Grid

Same rule as in static design: break the grid deliberately, after establishing it. A full-bleed element that ignores all grid relationships reads as intentional energy when it's one element in an otherwise structured composition. When everything ignores the grid, the composition is just chaotic.
