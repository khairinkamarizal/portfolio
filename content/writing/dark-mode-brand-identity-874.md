## Dark Mode Is Not Inversion

The most common mistake in adapting brand identities for dark mode is simple inversion — swapping light and dark values and calling it done. A brand palette designed for light backgrounds inverted to dark backgrounds doesn't produce a coherent dark mode identity. It produces an inverted light-mode identity, which usually looks wrong.

Dark mode brand design requires rethinking the color relationships for a fundamentally different visual context, not mechanically reversing them.

## Why Inversion Fails

Brand palettes designed for light backgrounds typically have a primary color that works against white or near-white surfaces. That primary color has been calibrated for contrast, vibrancy, and emotional register in a light context.

When placed against a dark background, that same color at the same value often reads differently — sometimes more vivid than intended, sometimes with different apparent contrast, sometimes in conflict with the dark background in ways that weren't present on light backgrounds.

Equally, a dark brand color that serves as an accent on light backgrounds becomes a background color in a naive dark mode inversion. It was never designed to be a background color.

## Building Proper Dark Mode Brand Colors

A proper dark mode brand palette starts from the brand's core color identity and develops dark-mode-specific values that maintain the same emotional register and brand recognition in a different visual context.

Practically, this means:

**Background colors**: typically very dark neutrals with slight color temperature derived from the brand palette — not pure black, which reads as a harsh void.

**Primary brand color**: usually needs to be slightly lighter and slightly less saturated than the light-mode version to maintain the same perceptual weight against a dark background.

**Text colors**: not pure white, which is harsh on dark backgrounds. Usually a warm or cool off-white derived from the brand's neutral palette.

**Supporting colors**: recalibrated for dark context, not inverted from light context.

## Logo Adaptation

Most logos need a dark mode variant — a version with adjusted colors or a reversed/white version designed specifically for dark backgrounds.

A logo with a dark wordmark on a light background needs either a white wordmark or a full-color version designed for dark backgrounds. Simply inverting isn't usually the right answer because the inversion changes the color relationships in the mark.

## Specifying Dark Mode in Guidelines

Brand guidelines need dark mode specifications with the same specificity as light mode: exact color values, approved logo variants, application examples. "Use the reversed logo on dark backgrounds" is not sufficient guidance when there are dozens of dark background scenarios with different brand color contexts.
