## References Are Not Moodboards

There's a difference between a reference and a moodboard. A moodboard is aesthetic inspiration — images that feel right, that capture a vibe. A typographic reference is a specific example of a decision I want to understand: why does this spacing work, how was this hierarchy achieved, what makes this pairing feel right.

I collect typographic references with intent. I don't just save things that look good. I save things that solve a problem I'm trying to solve, or that demonstrate a technique I want to understand better.

## What I Actually Analyze

When I study a typographic reference I'm asking specific questions:

**Scale relationships**: what is the ratio between the largest and smallest type? How far apart are the weights in the hierarchy?

**Spacing decisions**: how is the tracking set relative to the size? How much space separates typographic zones? Does the letter-spacing feel tight, normal, or open for the size?

**Typeface character**: what does the specific typeface bring to this context? Would a different face in the same weight change the feeling, and if so, how?

**Alignment logic**: is the type flush left, centered, or right-aligned? How does the alignment interact with the page structure?

**Color and value**: what values is the type set in? How does the type value relate to the background?

Asking these questions turns passive appreciation into active learning.

## Building a Personal Reference Library

I organize my references by problem type rather than by aesthetic. Folders for: condensed headline treatments, tight-tracking display, large-scale single-word posters, mix of serif and sans in the same layout, type used as texture.

When I'm working on a specific problem, I can go directly to the folder that addresses it. This is more useful than an undifferentiated collection of things I find attractive.

## Physical References

I collect printed material specifically for typographic study: magazines, books, exhibition catalogs, annual reports, packaging. Print reveals things digital references don't — ink behavior on paper, the physical relationship between type size and page size, how leading and spacing feel at actual scale.

I keep a physical file of pages pulled from print work that demonstrates something worth studying. These get looked at more carefully than digital references because the act of handling them slows down the looking.

## The Reverse Engineering Habit

When I encounter a typographic treatment I find particularly effective, I try to reverse-engineer it. Not to copy it — to understand the decisions. What size is this type approximately? What leading? What tracking? What typeface? Trying to answer these questions precisely forces close observation that builds the visual vocabulary I draw on in my own work.

Over time this habit produces a dense library of internalised knowledge about what works and why, which is ultimately more useful than any external reference file.
