## The Pattern as Brand Extension

Most brand identities include a logo, a color palette, and typography. The pattern is often an afterthought — or missing entirely. This is a missed opportunity.

A brand pattern is a surface-level identity element that can cover packaging, shopping bags, wrapping paper, fabric, backgrounds, presentation slides, and any number of other applications where the full logo would be too dominant or too small to be effective.

A strong pattern extends the brand's visual world into texture and rhythm. It makes the brand feel more considered, more complete, more real.

## What Makes a Pattern Feel Branded

Not every pattern is a brand pattern. A generic geometric grid or a stock pattern doesn't carry brand meaning. A pattern is branded when it's built from elements that belong exclusively to the brand's visual world.

These elements might be:

- Motifs derived from the logo form — sections of the mark, geometric elements from the logo's construction
- Shapes that reference the brand's product, industry, or values
- Typography from the brand's type system used as a repeating element
- Proprietary icons or illustration elements from the identity system

The test: could this pattern belong to any brand, or does it unmistakably belong to this one?

## The Three Pattern Structures

I work with three fundamental pattern structures depending on the brand's personality:

**Grid patterns** — elements arranged in a precise, regular grid. Reads as systematic, controlled, modern. Good for technical, financial, or precision-oriented brands.

**Half-drop patterns** — elements offset by half a unit on alternating rows, creating a diagonal flow. Reads as more dynamic and organic than a strict grid. The most versatile structure.

**Scattered patterns** — elements placed with varied spacing and sometimes varied orientation, following an irregular layout. Reads as playful, artisanal, organic. Good for lifestyle, food, or creative brands.

The structure should be chosen to match the brand's personality, not just because it looks interesting.

## Scale and Density

Scale and density are the primary variables I adjust once the base pattern is established.

A dense pattern at small scale creates a texture that reads from a distance as a solid color. Good for backgrounds that need to add subtle richness without visual complexity.

A loose pattern at large scale creates clearly identifiable motifs. Good for packaging where individual elements need to be read and recognized.

I typically develop the pattern at two or three scales: dense/small, medium, and large/featured. Different applications call for different scales.

## Color in Patterns

Patterns can work in multiple color configurations: the full brand palette, a monochromatic version in the primary brand color, and a reversed version (light on dark).

I design the base pattern in flat black on white, then test it across color configurations. Patterns that only work in one color combination are less versatile than patterns that work across multiple configurations.

## The Integration Test

A brand pattern is working when you can place it alongside the logo and the core typography and have all three feel like they came from the same designer, in the same session, with the same intent.

When the pattern feels like it was grabbed from a different project or a stock library, it's not doing brand work — it's just decoration.

Build patterns from brand DNA, not from pattern-making habit.
