## Type is Voice

When people talk about brand voice, they usually mean copy — the words a brand uses, the tone of its messaging. But visual type choices carry voice too. The typeface a brand uses communicates before a single word is read.

A humanist sans-serif says something different from a geometric one. A serif with visible calligraphic roots says something completely different from a modernist slab. Typography isn't neutral. It has personality, and that personality shapes how the brand is experienced.

## The Three-Tier System

Most functional brand type systems have three tiers: a display type used for headlines and large-scale applications, a body type used for paragraph-length text, and a utility type used for labels, captions, and interface elements.

Each tier has different requirements. Display type can be more expressive, more distinctive, because it's always used at sizes where its details read clearly. Body type needs to be highly legible at small sizes and comfortable over long reads. Utility type needs to be clean and functional — it's working in tight spaces with minimal formatting latitude.

## Contrast and Harmony

The relationship between the typefaces in a system needs to work on two levels simultaneously: they need to be distinct enough to create visual hierarchy and interest, and they need to feel like they belong together.

The classic pairing logic — serif and sans-serif from complementary design periods, or a neutral body type with an expressive display — works because the contrast is structural but the aesthetic sensibility is shared. Two wildly different typefaces from different eras and traditions rarely coexist comfortably.

I always test pairings together in context before committing. Typefaces that look fine individually can clash badly in combination.

## Sizing and Spacing Rules

A type system isn't just typeface selection — it includes a sizing scale (what sizes get used), spacing rules (how much room goes above and below different type elements), and weight usage guidelines (which weights appear where).

These rules create the visual rhythm of the brand. When they're followed consistently, any piece of brand communication — regardless of who designed it — feels like it belongs to the same family.

## Licensing is Real

One practical note that designers sometimes overlook: typeface licensing matters. A font licensed for desktop use may not be licensed for web embedding, app development, or broadcast use. Before committing a typeface to a brand system, I verify that the required licenses exist or can be obtained.

A beautiful type choice that the client can't legally use is worse than useless.

## When to Go Custom

For brands that can justify the investment, custom type is the strongest option. A typeface designed specifically for the brand is inherently unique and becomes a genuine brand asset. Nobody else can use it.

That's expensive. For brands at scale — and for brands where distinctiveness is a strategic priority — it's worth it.
